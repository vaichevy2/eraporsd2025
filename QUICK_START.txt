## 🛠️ IF SOMETHING GOES WRONG

### Problem: Can't add data
- Check console for error messages (F12)
- Verify form has required fields filled
- Try refreshing (F5) and trying again

### Problem: Data disappears after reload
- Open DevTools (F12) → Application → IndexedDB → RaporDeepLearningDB
- Check if data is stored in the stores
- Look for error messages in Console
- Try manually running: `await app.persistAllData()`

### Problem: Console shows errors
- Note the exact error message
- Check if it mentions specific store names
- Verify form inputs are correct types (strings, numbers)
- Check browser compatibility (use Chrome/Edge if possible)
